# Conditional-Probability-in-cryptography
Zgjidhja e Enigmave të Kriptografisë duke Përdorur Probabilitetin e Kushtëzuar

Permban 3 files:

encrypt.py - Skript e thjeshte që realizon enkriptimin me Vigenere. Perdoruesi jep input mesazhin dhe çelesin dhe merr output mesazhin e enkriptuar
decrypt.py - Permban te gjitha funksionet matematikore. Perdoruesi jep input mesazhin e enkriptuar dhe merr si output çelesin dhe mesazhin origjinal
gui.py - Ofron nderfaqe te userit per perdorimin e funksioneve paraprake.

Programi eshte realizuar si mjet studimi i projektit "Zgjidhja e Enigmave te Kriptografise duke perdorur probabilitetin e kushtezuar"

Per te provuar programin ekzekutoni:

python3 gui.py
